# TODO Tree for Unity


## Support Features

  1. Tag (TODO, FIXME)
  2. Priority (Minor, Medium, Major)
  3. Progress (Active, On-Hold, Completed)
  4. Reorder (Drag & Drop the todo)

You can add your custom settings in TodoConfig.asset. (+ color for each of them)
